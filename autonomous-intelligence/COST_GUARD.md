# Cost Guard

This repository work must stay within ordinary GitHub repository operations.

The following are disabled unless the owner explicitly approves them after being told the expected cost or quota impact:

- Git LFS
- GitHub Actions workloads
- GitHub Codespaces
- paid storage or bandwidth add-ons
- paid third-party artifact hosting
- any automatic billing upgrade

If a required artifact cannot fit in ordinary Git history, stop and register it as an external heavy artifact instead of enabling a paid service.
